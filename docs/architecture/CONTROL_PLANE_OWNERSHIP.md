# Control Plane Ownership

## registry-service
Owns:
- agent registration
- specialist routing
- contract lookup

## policy-service
Owns:
- policy validation
- operational rules
- governance enforcement

## risk-engine
Owns:
- risk scoring
- operational risk analysis

## approval-service
Owns:
- pending approvals
- approval history

## execution-service
Owns:
- execution orchestration
- task runners

## audit-service
Owns:
- immutable audit lineage
- execution records

## memory-service
Owns:
- operational memory
- outcomes
- checkpoints

## monitoring-service
Owns:
- health checks
- metrics
- alerts
